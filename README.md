# gRPC_Metrics
App using gRPC to collect and analyze metrics

 1. Get the code:

    ```console
    $ git clone github.com/vovaenjoy1/gRPC_Metrics
    ```

 2. Run the server:

    ```console
    $ go run server/server.go
    ```

 3. Run the client:

    ```console
    $ go run client/client.go
    ```
